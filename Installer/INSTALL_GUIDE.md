# Vectra Connect — Install Guide

**Time to install: ~2 minutes. No IT support needed.**

---

## Step 1 — Download

Click the link in the email (or SharePoint) to download **VectraConnect_Setup.exe**.

Save it anywhere — your Desktop is fine.

---

## Step 2 — Run the installer

Double-click **VectraConnect_Setup.exe**.

> If Windows shows a SmartScreen warning ("Windows protected your PC"), click
> **More info → Run anyway**. This is normal for new software from your organisation.

The setup wizard will open:

1. **Welcome** — click **Next**
2. **Licence Agreement** — read, tick *I accept*, click **Next**
3. **Ready to Install** — click **Install**
4. **Finished** — click **Finish**

The installer will automatically install any missing components (.NET 4.8, Office runtime).
If it needs to download something, this may take 1–2 extra minutes on slower connections.

---

## Step 3 — Use it

1. Open **Microsoft Project** (close and reopen it if it was already running).
2. Click the **Task** tab in the ribbon.
3. Find the **Vectra Connect** group on the right side of the ribbon.
4. Click **Publish Schema**.
5. The first time, you'll be asked to choose an output folder — pick any folder you like.
6. Your Excel file (and optional CSVs) will appear there instantly.

---

## Troubleshooting

| Problem | Fix |
|---------|-----|
| Button doesn't appear in ribbon | Close and reopen MS Project. If still missing, run the installer again. |
| "Windows protected your PC" warning | Click More info → Run anyway |
| Installer asks for admin password | Click Yes — it only needs this for the .NET/Office runtime, not the add-in itself |
| Export fails with an error | Make sure a project file is open in MS Project before clicking Publish |

---

## Uninstall

Go to **Windows Settings → Apps → Vectra Connect → Uninstall**.
